# SDCard_module_DIPSPI
這是一個和 Taiwanduino 相容的 microSD 記憶卡擴充板  
他可以直接取代原本所使用的 25AA512 EEPROM 記憶體

**使用說明**
* TaiwanDuino :   
 [/software/Taiwanduino/example][sw] 資料夾內的程式碼使用 Arduino IDE 開啟  
 預設播放檔名為 0.wav ~ 7.wav 的 PCM 編碼音訊檔  
 [安裝相對應的程式庫][libs]之後即可編譯上傳
